# Blev
Imagina um único local com vários artigos, vídeos e novidades sobre a carreira de tecnologia esse local é o Blev. A intenção é invez do usuário acessar vários sites procurando novos assuntos, ele vai encontrar todos aqui nesse blog e ser redirecinado ao site do assunto. Então chegue de ficar com várias abas abertas procurandos novidades, o Blev faz isso para você.
